## What is BMS? 🏗️

- A Building Management System (BMS) is an integrated control system used to manage and automate building subsystems such as:

- Lighting

- HVAC

- Shutters

- Sensors

- Energy management

- Modern BMS architectures rely on standard communication protocols to ensure scalability, reliability, and interoperability.

---

## What is KNX? 🌐

- KNX is an international standard protocol for building automation.

- It is based on distributed intelligence, meaning:

- Devices operate independently

- Logic can be implemented inside the KNX system

- Communication is handled through Individual Addresses and Group Addresses

- It supports KNX TP and KNX/IP

- In this repository, KNX forms the core automation backbone.
